在hexo中添加algolia搜索
2020-03-16|Hexo

mark 一下怎么将 algolia 添加到自己的主题，成功使用搜索功能，首先我引用的主题配置文件里已经有相关 algolia 的配置，

默认为 false，我主要说在把值改为 true 的后续操作。

  1. 首先需要获取 Algolia 官网 注册获得了 applicationID 和 apiKey

注册完成之后，点击 Create Index 新建一个索引

然后到菜单 API Keys 那里的 All API Keys，点击 New API Key 新生成一个 API Key，并在 ACL 那里根据个人需求选择权限

这里的 applicationID 等信息后面会用到


  1. 安装 hexo-algolia

运行 npm install --save hexo-algolia ，在你的项目里安装 hexo-algolia
建议浏览 hexo-algolia 文档

  1. 配置 algolia 信息

在 _config.yml 里添加前面提到你的 algolia 信息，

之后运行 hexo algolia ，你会看到错误提醒

看过官方文档，你还需要运行
$ export HEXO_ALGOLIA_INDEXING_KEY=你的apiKey
之后再运行 hexo algolia ，你就会看到成功提示

去 algolia 官网你就会看见你的文章数据已经添加到你的应用索引了

  1. 在官网你的 index 配置下设置你搜索的字段

到这里，你应该就可以在你的 hexo 里面使用搜索功能了。
